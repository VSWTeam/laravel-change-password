```bash
php artisan password:change --account=<account> --password=<password>
```

## Installation

You can install the package via composer:

```bash
composer require vswteam/laravel-change-password
```

## Usage

```bash
php artisan password:change <account> <password>
```

## Testing

``` bash
composer test
```
